# Todo List App

This is a simple Todo List web application built using HTML, CSS, and JavaScript with the help of an AI-powered vibe coding tool.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Persistent storage using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## Setup Instructions
1. Clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.

The application works entirely in the browser and requires no additional setup.
